# claude-skills

Custom Claude.ai skills built to solve real problems.

---

## 🧠 lean-reply

**Token-efficient response mode for Claude.**

### What it does
Forces Claude to give accurate, precise answers using the fewest tokens possible — so you can work longer without hitting your usage limit.

### Rules it enforces
- No filler, preamble, or closing remarks
- Answers start immediately, end when done
- Bullet points kept to 1 line unless detail is critical
- Code is clean — no unnecessary comments
- Accuracy is never sacrificed for brevity

### Install
1. Download `lean-reply.skill`
2. Go to [claude.ai](https://claude.ai) → Settings → Skills
3. Upload the file
4. Active in every future conversation

### Trigger phrases
Works automatically when you say things like:
- "use less tokens"
- "be brief / be concise"
- "save my tokens"
- "I'm on a free plan"
